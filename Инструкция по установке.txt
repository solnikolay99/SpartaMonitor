1) Скачиваем open-jdk 21 (https://download.java.net/java/GA/jdk21.0.2/f2283984656d49d69e91c558476027ac/13/GPL/openjdk-21.0.2_windows-x64_bin.zip)
2) Распаковываем в любое удобное место
3) Открываем "Настройки > Система > О системе" (Win + Pause), кликаем на "Дополнительные параметры среды > Параметры среды"
4) В системных переменных добавляем переменную JAVA_HOME и прописываем путь до bin папки, куда распаковали jdk (например C:\Users\openjdk-21.0.2\bin)
5) В системную переменную Path добавляем ссылку на переменную %JAVA_HOME%
6) Перезагружаем систему. Проверяем в командной строке правильность установки командой java --version - должно выдать номер установленной java
7) Распаковываем файл SpartaMonitor.zip в папку SpartaMonitor
8) Запускаем файл SpartaMonitor.bat
